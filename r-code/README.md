
Usage
---

Once you collected the data on ~/data/nao2, run the script on R typing:    

      $ source('~/plot_rawdata_2sensors.R', echo=FALSE)   


IMPORTANT VARIABLES:

* data_path_name <- "nao2"
* TAGIMU <- "imu1" or TAGIMU <- "imu3"
* TwoSensorAxis <- 4
* TwoSensorWindowFrame <- 000:1000
